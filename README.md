## 🎨 **Customize**

```
📁 src/
  ├── 🎨 styles/     # CSS files
  ├── ⚡ scripts/    # JavaScript
  └── 📸 assets/     # Images & certificates
```

- Replace images in `assets/`
- Edit content in `index.html`  
- Modify colors in `styles/main.css`
